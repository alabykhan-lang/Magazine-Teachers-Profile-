# 📖 School Magazine — Teacher Profile System

## Files in this package
- `admin.html` — YOUR settings panel (configure fields, photo rules)
- `form.html` — Teacher-facing submission form (share this link)
- `submissions.html` — View & export all submissions

---

## 🚀 HOW TO HOST FOR FREE (Netlify — 5 minutes)

### Step 1: Go to netlify.com
Open https://netlify.com and create a free account (use your email).

### Step 2: Deploy
- On your Netlify dashboard, look for **"Deploy manually"** or drag-and-drop
- Drag the entire `magazine-form` folder into the Netlify deploy zone
- Netlify gives you a free link like: `https://random-name-123.netlify.app`

### Step 3: Share
- Your form link will be: `https://your-site.netlify.app/form.html`
- Your admin link: `https://your-site.netlify.app/admin.html`
- Your submissions: `https://your-site.netlify.app/submissions.html`

---

## ⚙️ ADMIN PANEL FEATURES
- Toggle each field ON or OFF
- Mark fields as Required or Optional
- Set photo minimum size (width × height in pixels)
- Set maximum file size (1–5 MB)
- Choose accepted formats (JPG, PNG, WebP)
- Add school name, edition title, deadline, contact number

## 📋 FIELDS AVAILABLE
1. Full name
2. Subject(s) taught
3. Years at school
4. Highest qualification
5. Message to students
6. Fun fact / Hobbies
7. Nickname
8. Notable achievement
9. Favourite quote
10. Career advice to students
+ Profile photo (with full validation)

## 📤 EXPORTING TO CLAUDE
1. Go to `submissions.html`
2. Click **"Export all for Claude"**
3. A `.txt` file downloads with all profiles neatly listed
4. Upload that file to Claude — profiles get formatted instantly
